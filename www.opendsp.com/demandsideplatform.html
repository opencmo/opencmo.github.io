Demand Side Platform

Demand Side Platform(DSP), the system that connects to multiple ad exchanges and other media suppliers, accessed by a single user interface.

This system will provide the interface for advertiser to buy multiple real-time ad exchange with Real Time Bidding.

DSP features like universal frequency capping, robust auditing, content filtering, and context targeting and targeting based on users demographic details.

Publisher can sell each impression for advertiser’s ads. They can set minimum bid rate for their zone.This system will sell the impression when only the advertiser bid rate meet the minimum bid rate set by publisher. Otherwise publisher can deliver their own ads.

Key Benifits :

Access to multiple ad exchanges through a single interface.
Contextual Targeting :

Delivering advertiser banners with the relavent context availble publisher site. (For example : Advertiser 1 can bid their ad should shown to the Sports context relavent publisher site. Their ad will deliver only to the sports site.

From this advertiser can deliver ads to right auidence.

Re-Targeting :

Re-Target existing user for the advertiser. (For Example : Advertiser1 can bid for “This user has made conversation earlier on my site”. If this advertiser wins the auction , that particular advertiser ad will shown to that user.)

From this, can re-target advertiser existing user who made visit on their site.
